This is homework for Data Communication and Network, 2014 fall

### Usage

* Start the server

    `$ python server.py <port>`

    For example:

    `$ python server.py 9999`

* Start the client

    `$ python client.py <ip> <port> | <ip:port>`

    For example:

    `$ python client.py 127.0.0.1 9999`

    or

    `$ python client.py 127.0.0.1:9999`
